# Carpeta de Fondos

Esta carpeta contiene las imágenes de fondo para las recompensas de la tienda.

## Archivos requeridos:

- **mexico.jpg** o **mexico.png** - Fondo de perfil con los colores de México

## Instrucciones:

1. Coloca tu imagen en esta carpeta
2. Nómbrala exactamente: `mexico.jpg` o `mexico.png`
3. Dimensiones recomendadas: 1920x1080px o patrón de 400x400px
4. El fondo se aplicará automáticamente al activar la recompensa "Fondo de Perfil: México"
